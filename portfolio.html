HioMail

API de serviço de envio mensagens por email para websites, aplicativos mobile ou desktop integrável com qualquer linguagem de programação.

HioMail é um serviço que podes integrar num site ou aplicativo que estejas a desenvolver. Supondo que queiras deixar um formulário para os usuários(visitantes) poderem contactar a entidade dona do site/app sem sair para usar outros serviços como Gmail etc. HioMail tira de ti o trabalho de ter que configurar essa funcionalidade sempre. Basta fazer uma requisição para a API com os dados do visitante, dados da mensagem e dados da entidade dona do site. HioMail se encarrega de enviar a mensagem do visitante para a entidade e ainda de mandar uma resposta automática ao visitante. Resposta essa que pode ser personalizada.
